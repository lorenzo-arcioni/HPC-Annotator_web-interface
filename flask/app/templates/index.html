Welcome to HPC-T-Annotator

Empowering Annotation Workflows

HPC-T-Annotator is a cutting-edge tool designed to automate the generation of annotation software for de novo transcriptome assembly, utilizing advanced parallelization techniques.

Our interface is designed to dynamically generate a parallel annotation software package, optimized for HPC clusters and requiring no user expertise in parallel programming or specialized IT skills.

HPC-T-Annotator integrates third-party software like BLAST or DIAMOND for parallel annotation, and allows users to configure the utilization of computational resources via a web interface, allowing control over the number of processes deployed.

Upon user-driven customization, the software package is ready for deployment. It can be downloaded in TAR format and transferred to the target machine for optimized, scalable annotations.
