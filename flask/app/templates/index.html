Welcome to HPC-T-Annotator

Empowering Annotation Workflows

HPC-T-Annotator is a cutting-edge tool designed for the fully automated generation of annotation software for de novo transcriptome assembly, leveraging powerful parallelization techniques.

Our interface has been architecturally developed to dynamically generate a parallel annotation software package. This package is optimized for execution on HPC clusters, obviating the need for users to possess expertise in parallel programming or specialized IT competencies.

For parallel annotation, HPC-T-Annotator employs third-party software such as BLAST or DIAMOND. Its capacity to utilize extensive computational resources depending on the number of processes configured by the user via the web interface.

Upon user-driven generation of the customized software package, it becomes ready for deployment. This package can be readily downloaded in TAR format and disseminated to the target machine for optimized and scalable annotations.
